CSS Feature: :has() Pseudo-Class

The :has() pseudo-class is a powerful new addition to CSS that allows you to select an element based on its children or descendants.

Example:


.container:has(.child) {
  border: 2px solid blue;
}

In this example, the .container element will have a blue border if it contains an element with the class .child.

Use this feature to style elements dynamically based on their content or structure.
